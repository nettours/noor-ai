📂 محتويات هذه الحزمة:

  📁 frontend/              → ملفات Next.js
     ├─ vercel.json         → إعدادات Vercel للنشر
     ├─ manifest.json       → ضع في public/manifest.json
     ├─ sw.js               → ضع في public/sw.js (للـ PWA)
     ├─ SWRegister.tsx      → ضع في src/components/ui/
     ├─ layout.tsx          → استبدل في src/app/layout.tsx
     └─ .env.example        → انسخ كـ .env.local

  📁 backend/               → ملفات Node.js
     ├─ railway.toml        → إعدادات Railway
     ├─ package.json        → استبدل القديم
     ├─ tsconfig.json       → استبدل القديم
     └─ .env.example        → انسخ كـ .env

  📁 docs/
     └─ DEPLOYMENT_GUIDE.txt → الدليل الشامل بالعربية

  📄 deploy.bat             → سكريبت نشر تلقائي
  📄 .gitignore             → لحماية الأسرار من GitHub

═══════════════════════════════════════════════════════════════════
🚀 طريقة الاستخدام (3 طرق):
═══════════════════════════════════════════════════════════════════

──────────────────────────────────────────────────
💫 الطريقة 1: التلقائية (الأسهل)
──────────────────────────────────────────────────

1. ضع الملفات في G:\noor-ai
2. اضغط مرتين على: deploy.bat
3. اتبع التعليمات على الشاشة

──────────────────────────────────────────────────
🎯 الطريقة 2: اليدوية (التحكم الكامل)
──────────────────────────────────────────────────

اتبع: docs/DEPLOYMENT_GUIDE.txt

5 مراحل واضحة:
  ① تجهيز الكود ورفعه على GitHub
  ② نشر Backend على Railway
  ③ نشر Frontend على Vercel
  ④ ربطهما معاً
  ⑤ نطاق خاص (اختياري)

──────────────────────────────────────────────────
⚡ الطريقة 3: السريعة (إذا كنت محترف)
──────────────────────────────────────────────────

cd backend && railway up
cd frontend && vercel --prod
