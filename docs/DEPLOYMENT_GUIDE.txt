المرحلة 1: تجهيز الكود ورفعه على GitHub
═══════════════════════════════════════════════════════════════════

📦 الخطوة 1.1 — أنشئ حساب GitHub
   اذهب إلى: https://github.com/signup

📦 الخطوة 1.2 — أنشئ مستودع جديد
   1. اضغط الزر الأخضر "New repository"
   2. الاسم: noor-ai
   3. اختر Private (خاص) أو Public (عام)
   4. لا تضع أي خيار آخر
   5. اضغط "Create repository"

📦 الخطوة 1.3 — ارفع الكود

   افتح Terminal في G:\noor-ai واكتب:

   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/nettours/noor-ai.git
   git push -u origin main

   (استبدل USERNAME باسم حسابك)

📦 الخطوة 1.4 — تأكيد
   عُد لـ github.com وتأكد أن الكود ظهر

═══════════════════════════════════════════════════════════════════
المرحلة 2: نشر Backend على Railway
═══════════════════════════════════════════════════════════════════

⚡ الخطوة 2.1 — أنشئ حساب
   اذهب إلى: https://railway.app
   اضغط "Login" واختر "Login with GitHub"

⚡ الخطوة 2.2 — أنشئ مشروع جديد
   1. اضغط "New Project"
   2. اختر "Deploy from GitHub repo"
   3. اختر مستودع "noor-ai"
   4. اختر المجلد: backend

⚡ الخطوة 2.3 — اضبط متغيرات البيئة
   في dashboard المشروع → اضغط Service → Variables → Raw Editor
   انسخ هذه القيم وعدّلها:

   NODE_ENV=production
   PORT=4000
   JWT_SECRET=noor-jwt-secret-2025-CHANGE-ME-RANDOM-LONG-STRING
   JWT_REFRESH_SECRET=noor-refresh-2025-CHANGE-ME-DIFFERENT-RANDOM
   ANTHROPIC_API_KEY=sk-ant-XXXXX (اختياري)
   FRONTEND_URL=*

   اضغط "Update Variables"

⚡ الخطوة 2.4 — احصل على رابط Backend
   1. اضغط Service → Settings → Networking
   2. اضغط "Generate Domain"
   3. سيظهر رابط مثل: noor-backend-production.up.railway.app
   4. انسخه واحفظه ⭐

⚡ الخطوة 2.5 — اختبر Backend
   افتح: https://YOUR-BACKEND.up.railway.app/health
   يجب أن يظهر: {"status":"ok","users":0}

═══════════════════════════════════════════════════════════════════
المرحلة 3: نشر Frontend على Vercel
═══════════════════════════════════════════════════════════════════

⚡ الخطوة 3.1 — أنشئ حساب
   اذهب إلى: https://vercel.com/signup
   سجّل بـ "Continue with GitHub"

⚡ الخطوة 3.2 — استورد المشروع
   1. اضغط "Add New" → "Project"
   2. اختر "noor-ai" من القائمة
   3. اضغط "Import"

⚡ الخطوة 3.3 — اضبط الإعدادات
   - Framework Preset: Next.js (تلقائي)
   - Root Directory: اضغط Edit واختر: frontend
   - Build Command: npm run build
   - Install Command: npm install --legacy-peer-deps

⚡ الخطوة 3.4 — أضف متغيرات البيئة
   اضغط Environment Variables وأضف:

   اسم المتغير: NEXT_PUBLIC_BACKEND_URL
   القيمة: https://YOUR-BACKEND.up.railway.app/api
   (استخدم رابط Railway من المرحلة 2.4)

   اسم المتغير: NEXT_PUBLIC_APP_URL
   القيمة: https://noor-ai.vercel.app
   (سيتغير بعد النشر)

⚡ الخطوة 3.5 — انشر
   اضغط "Deploy"
   انتظر 2-3 دقائق ⏳

⚡ الخطوة 3.6 — احصل على الرابط
   سيظهر رابط مثل: https://noor-ai-xxxxx.vercel.app
   اضغط "Visit" لرؤية موقعك على الإنترنت! 🎉

═══════════════════════════════════════════════════════════════════
المرحلة 4: ربط Backend بـ Frontend
═══════════════════════════════════════════════════════════════════

⚡ الخطوة 4.1 — حدّث Frontend URL في Backend
   عُد لـ Railway → Backend → Variables
   عدّل: FRONTEND_URL=https://noor-ai-xxxxx.vercel.app
   احفظ التغيير

⚡ الخطوة 4.2 — حدّث APP_URL في Vercel
   اذهب لـ Vercel → noor-ai → Settings → Environment Variables
   عدّل: NEXT_PUBLIC_APP_URL لرابط Vercel النهائي
   اضغط Redeploy

═══════════════════════════════════════════════════════════════════
المرحلة 5: نطاق خاص (اختياري - $10/سنة)
═══════════════════════════════════════════════════════════════════

⚡ الخطوة 5.1 — اشترِ نطاق
   من Namecheap أو Octenium (عندك بالفعل)

⚡ الخطوة 5.2 — في Vercel
   Settings → Domains → Add
   مثلاً: noor.snetprodz.com

⚡ الخطوة 5.3 — في DNS Provider (Octenium Panel)
   أضف A record:
     Name: noor
     Type: CNAME
     Value: cname.vercel-dns.com

⚡ الخطوة 5.4 — انتظر 1-24 ساعة
   Vercel سيُفعّل HTTPS تلقائياً ✅

📱 تثبيت كتطبيق هاتف (PWA)
═══════════════════════════════════════════════════════════════════

بعد النشر، أي شخص يفتح موقعك من الهاتف يستطيع:

في Chrome على Android:
   1. افتح الموقع
   2. اضغط ⋮ (القائمة)
   3. اختر "إضافة إلى الشاشة الرئيسية"
   4. يظهر كأيقونة تطبيق! 🎉

في Safari على iPhone:
   1. افتح الموقع
   2. اضغط زر المشاركة 📤
   3. اختر "إضافة إلى الشاشة الرئيسية"

🆘 حل المشاكل الشائعة
═══════════════════════════════════════════════════════════════════

❌ Backend لا يعمل في Railway
   ✅ تحقق من Logs في Railway dashboard
   ✅ تأكد أن package.json يحوي "start": "node dist/server.js"
   ✅ تأكد أن JWT_SECRET موجود

❌ Frontend لا يتصل بـ Backend
   ✅ تحقق أن NEXT_PUBLIC_BACKEND_URL صحيح في Vercel
   ✅ افتح Console (F12) وانظر الأخطاء
   ✅ تأكد أن /health يعمل

❌ المكالمات لا تعمل
   ✅ يجب أن يكون الموقع HTTPS (Vercel يوفّره)
   ✅ تأكد من السماح بـ Camera + Microphone
   ✅ لاتصال عبر شبكات مختلفة قد تحتاج TURN server

❌ الصوت لا يعمل
   ✅ بعض CDNs محجوبة من بعض البلدان
   ✅ جرّب قارئاً مختلفاً من الإعدادات
